# Diode
### To compile (with g++):
g++ main.cpp -o rt -Wall -O3

### To run:
-./rt (args)

### Args:
- -i: info on args
- -d: enable debug
- -o <name>: specify output file name (without extention)
- -h <int>: specifiy image height
- -r <int> specify preset ratio (4 = 4:3, 16 = 16:9, 10 = 16:10, 1 = 1:1)
